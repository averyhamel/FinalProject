## NCAA Baseball Predictive Models
In this project, I will see how a team's roster continuity correlates with their year-over-year change in win percentage. With this, I hope to be able to create a model that will allow us to predict how a team will perform in the upcoming year based on the previous year's statistics and results and the upcoming season's statistics from incoming players (will have to find out if DII, DIII, NJCAA statistics should/will have to be weighed differently).
